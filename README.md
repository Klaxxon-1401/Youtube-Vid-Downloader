<h1>A simple Youtube video downloader</h1> 
<p>Currently only capable of downloading 1080p videos with Embedded Subtitles.</p>
<p>Future updates will include a subtitle toggle and the option to choose your preffered quality.</p>
<h3>Installation Instructions</h3>
<p>For the program to run properly the following dependancies <h7 style="font-weight: 700;">MUST</h7> be installed <br>1.ffmpeg <br>2.Phantomjs <br>3.yt-dlp <br>4.Python.</p>
<p>To install these on Windows.</p>

``` bash
winget install ffmpeg

winget install python3

python -m pip install -U yt-dlp

https://bitbucket.org/ariya/phantomjs/downloads/phantomjs-2.1.1-windows.zip #download,extract the files and run the exe
```
<p>To install these on Linux (Arch and Arch-Based).</p>


```bash


sudo pacman -S ffmpeg python

sudo pacman -S yt-dlp

sudo pacman -S phantomjs
```
<p>Run the afforementioned commands in CMD or Terminal.</p>
<p>For Windows users who get a Smartscreen dialog box , simply click "More info" and then "Run Anyway".</p>
